CMD Center of File Server

Currently Running on Port: %1 | Current File Access Port: %2 | Current Msg Server: %3
All files are encrypted with Base64 and all messages are encoded with a custom key protocol

Current File Hosted: %7 

Use this format to input commands "http://ipaddress:port/command=PutCommandHere???parameter1???parameter

- getFile(filename):

   Sets the currently hosted file.
   filename - The directory of the file to be hosted (ex.) DIR\FILE (format) DIR+++FILE 

-download(website, dir+name);  

   Downloads file from website.
   website - A website hosting raw data | dir+name - The directory and name the file is saved as     

-mkDir(directory):

    Makes a directory at location
    directory - The directory to create and its location (ex.) FileServerFiles+++memes 

-delDir(directory):

   Deletes a Directory and its Children
   directory - The directory to delete (ex.) FileServerFiles+++memes            

-delFile(dir+filename):

   Deletes a file              
      dir+filename - The Directory and filename to be deleted (ex.) FileServerFiles+++memes+++image.jpg 

-dir(directory):  

   Shows selected directory info

-getFileRaw(filename):

    Returns data as ascii
